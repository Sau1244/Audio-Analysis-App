# Audio-Analysis-App

## Description

This project is an audio analysis application developed as part of a university course. It allows users to analyze audio files or microphone recordings, detecting is the file real or not, by AI.
The application includes a graphical interface and real-time analysis features.

---

## Features

- Load and analyze audio files
- Play loaded files
- Display spectrogram and waveforms graphs
- Display audio and AI results in GUI
